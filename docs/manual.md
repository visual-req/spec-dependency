## 使用手册

### Web 页面

启动服务后，浏览器打开终端输出的 URL（默认 `http://127.0.0.1:8766/`）。

页面包含两个入口：
- 识别依赖
- 依赖列表

### 识别依赖

1) 点击“选择文件”，选择一个或多个 `.doc/.docx`
2) 点击“开始识别”
3) 识别中会显示进度条
4) 识别完成后以表格展示结果（文件/模块/系统/API 数/置信度/证据）

结果说明：
- 文件名旁边会显示开始时间、结束时间、时长（本地化格式）
- 点击“API 数”会打开右侧抽屉查看命中的 API

抽屉说明：
- 支持翻页
- 左侧树形目录（按 path 第一段分组）展示 API 清单
- 点击 API 显示右侧详情（method/path/keywords/置信度/证据）
- 点击遮罩层或按 ESC 关闭

### 依赖列表

切换到“依赖列表”后：
- 可点击“刷新”重新加载依赖清单
- 表格中点击“API 数”打开抽屉查看 API 清单

### 配置项

配置通过 `SPEC_DEP_CONFIG` 指定配置文件路径（yaml/json）。当前支持：
- `server.host` / `server.port`
- `work_dir`
- `llm.base_url` / `llm.model` / `llm.api_key`
